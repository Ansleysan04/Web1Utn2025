Contenido Principal

El posicionamiento en CSS es una herramienta fundamental para controlar la disposición de los elementos en una página web. Nos permite sacarlos del flujo normal del documento y ubicarlos de manera precisa para crear diseños complejos y atractivos.

Esta sección principal tiene aplicado el posicionamiento relative. Esto significa que inicialmente se comporta como un elemento estático, ocupando su lugar en el flujo del documento. Sin embargo, al aplicar propiedades como top, right, bottom o left, podemos desplazarlo relativamente a su posición original. Una de las utilidades clave del posicionamiento relativo es establecer un punto de referencia para elementos con posicionamiento absolute dentro de él.

La imagen que ves aquí está dentro de un contenedor con posicionamiento relative. La propia imagen tiene aplicado el posicionamiento absolute. Esto la saca del flujo normal del documento y la posiciona relativamente al primer ancestro que tenga un posicionamiento diferente a static (en este caso, el div.contenedor-imagen). Al usar top: 0; y left: 0;, la hemos colocado en la esquina superior izquierda de su contenedor.

Sección con Caja Sticky

Esta sección demuestra el uso del posicionamiento sticky. El elemento con la clase sticky-box se comportará inicialmente como un elemento con posicionamiento relative dentro de esta sección.

Caja Sticky

¡Soy una caja con posicionamiento sticky!

Cuando hagas scroll y mi posición alcance los 20 píxeles desde la parte superior de mi contenedor, me "pegaré" a esa posición hasta que el contenedor deje de ser visible.

El posicionamiento sticky es útil para crear elementos que permanecen visibles mientras el usuario navega por una sección específica de la página, como menús de navegación secundarios o indicadores de progreso.
